# 綠界金流系統

## 描述
第三方金流系統。處理付款交易並透過回調通知本系統付款結果。同時提供發票開立、作廢與折讓 API。

## 關鍵屬性
- 第三方系統（非自有系統，屬合法 Actor）
- 透過 HTTP POST 回調通知付款結果（REQ-005）
- CheckMacValue SHA256 驗證（REQ-005）
- 提供發票開立、作廢、折讓 API（REQ-006, REQ-009, REQ-010）
- 綠界全方位金流 API v5（REQ-005）

## 來源
- REQ-005: 金流回調（付款成功/失敗）
- REQ-006: 發票開立 API
- REQ-009: 發票作廢 API
- REQ-010: 發票折讓 API
