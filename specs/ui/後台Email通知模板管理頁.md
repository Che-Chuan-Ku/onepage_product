# 後台Email通知模板管理頁

## 描述
管理員編輯系統 Email 通知模板的後台頁面。支援下單確認通知、付款成功通知、付款失敗通知與發貨通知四種固定模板的自訂。

## 佈局
- 模板列表（四筆固定項目）：
  - 下單確認通知模板
  - 付款成功通知模板
  - 付款失敗通知模板
  - 發貨通知模板
- 編輯區：
  - 模板標題輸入欄位
  - 模板內文編輯區（HTML 模板，結構固定）
  - 可用變數說明區：以表格或列表方式清楚呈現所有可用變數及其中文說明（REQ-032）：
    | 變數 | 說明 |
    |------|------|
    | `{{customerName}}` | 顧客姓名 |
    | `{{orderNumber}}` | 訂單編號 |
    | `{{totalAmount}}` | 訂單總金額 |
    | `{{websiteName}}` | 網站名稱（REQ-032 新增） |
    | `{{contactInfo}}` | 店家聯絡資訊（REQ-032 新增） |
  - 預覽按鈕
  - 儲存按鈕

## 互動行為
- 點擊模板項目切換編輯區內容
- 模板僅支援編輯，不支援新增或刪除
- 內文支援預設變數替換，變數以 {{變數名}} 格式標示
- `{{websiteName}}` 替換為所屬網站名稱，`{{contactInfo}}` 替換為網站 Footer 副標題內容（REQ-032）
- 預覽功能以範例資料填入變數顯示最終效果（含新變數的替換效果）（REQ-032）
- 儲存後即時生效，下次觸發通知時使用新模板
