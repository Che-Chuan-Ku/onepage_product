# 後台使用者管理頁

## 描述
管理員管理系統使用者的頁面。僅管理員可見此頁面，一般使用者不顯示此 MENU 項目。

## 佈局
- 頁面頂部：「新增使用者」按鈕（REQ-033）
- 使用者清單表格：使用者名稱、Email、角色下拉選單（管理員/一般使用者）
- 角色變更即時生效
- 新增使用者表單（模態框或獨立頁面）：（REQ-033）
  - Email（必填）
  - 姓名（必填）
  - 密碼（必填，8碼以上）
  - 角色選擇（管理員/一般使用者）
  - 是否寄送歡迎 Email（勾選框）

## 互動行為
- 下拉選單切換角色
- 不可將最後一位管理員降級（操作時提示警告）
- 點擊「新增使用者」按鈕開啟新增表單（REQ-033）
- 新增成功後即時出現於使用者清單（REQ-033）

## 來源
