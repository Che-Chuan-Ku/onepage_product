# 後台網站管理頁

## 描述
管理員建立、編輯、啟停網站的管理頁面。以清單 + 編輯面板單頁佈局呈現。

## 佈局
- 頁面頂部：返回上一頁按鈕（REQ-028）
- 左側：網站清單（含狀態標籤：草稿/已上線/已下線）
  - 已下線網站卡片顯示「重新上線」按鈕（REQ-028）
- 右側：編輯面板
  - 網站名稱（必填）
  - 網站標題（必填，前台頁面顯示用）（REQ-029）
  - 網站副標題（選填）（REQ-029）
  - 瀏覽器分頁標題 `<title>` tag（選填）（REQ-029）
  - 訂閱方案（文字描述）
  - 上架時間 / 下架時間
  - Banner 主視覺圖片上傳（必填，建議尺寸標示）（REQ-029）
  - 宣傳文案圖片上傳（選填）（REQ-029）
  - Footer 標題（選填，供店家填寫店名）（REQ-029）
  - Footer 副標題（選填，供店家填寫聯絡資訊與說明文字）（REQ-029）
  - 滿額免運門檻設定（預設 NT$1,500）
  - 上架商品選擇區（從商品清單中勾選要上架至此網站的商品 + 設定各商品上架時間）
  - 網站前台連結（自動產生 /project/{網站ID}）

## 互動行為
- 點擊清單項目切換編輯面板
- 啟用/停用按鈕控制網站狀態
- 已下線網站顯示「重新上線」按鈕，點擊後需二次確認，確認後狀態變為已上線（REQ-028）
- 上架商品選擇時可批次勾選商品並設定各商品上架時間（商品本身不記錄網站歸屬，關聯由網站端管理）
- 自動產生前台 URL
- 儲存後前台所有設定（標題/副標題/Banner/Footer等）即時生效（REQ-029）
- 返回按鈕導向上一頁（REQ-028）
