# 前台商品集合頁

## 描述
顧客透過 /project/{網站ID} 進入的商品集合頁面。一頁式滾動展示。

## 佈局（由上至下）
1. 瀏覽器分頁標題（`<title>` tag，對應後台設定的瀏覽器分頁標題）（REQ-029）
2. 網站 Banner 主視覺圖片（REQ-029）
3. 宣傳文案圖片（選填，無上傳則不顯示）（REQ-029）
4. 網站標題與副標題（對應後台設定）（REQ-029）
5. 商品列表（卡片式排列）（REQ-030）
   - 每張卡片：商品圖片、名稱、價格
   - 點擊卡片進入 /project/{網站ID}/product/{商品slug} 商品詳情頁
   - 顯示所有已到達上架時間且已關聯至本網站的商品（REQ-030）
6. Footer（REQ-029）
   - Footer 標題（對應後台設定，如店名）
   - Footer 副標題（對應後台設定，如聯絡資訊）
   - **OnePage 平台區（REQ-035）**：與店家 Footer 有視覺區隔線；靜態文字「由 OnePage 提供技術支援」+ 「聯絡我們」連結；點擊連結以新分頁開啟外部 OnePage 官網

## 互動行為
- 僅顯示已到達上架時間且網站狀態為「已上線」的商品（REQ-030）
- 響應式設計支援手機與桌面瀏覽
- FCP < 3 秒
