# 前台商品詳情頁

## 描述
顧客透過 /project/{網站ID}/product/{商品slug} 進入的商品詳情頁面。

## 佈局（由上至下）
1. 圖片輪播
2. 商品名稱
3. 價格（預購商品顯示折扣後價格 + 原價刪除線）
4. 包裝選項
5. 加入購物車按鈕
6. **OnePage 平台 Footer（REQ-035）**：靜態文字「由 OnePage 提供技術支援」+ 「聯絡我們」連結（新分頁）；與頁面內容有視覺區隔線

## 互動行為
- 不顯示庫存數量
- 預購商品動態顯示預購標籤與倒數計時器（伺服器時間 API 同步 + 前端本地倒數）
- 組合包額外顯示包含商品列表與折扣後價格
- 響應式設計
- FCP < 3 秒
