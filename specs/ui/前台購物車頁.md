# 前台購物車頁

## 描述
顧客的購物車獨立頁面（/cart）。

## 佈局
頁面版面須與 prototype 設計一致（REQ-031），由上至下：
- 商品列表：商品圖片（左）、名稱與單價（中）、數量調整（+/-）、刪除按鈕（右）、小計
- 運費預估顯示（顯示距免運差額或已達免運）
- 小計與總計金額顯示區（右下對齊）
- 前往結帳按鈕（固定於頁面底部）
- **OnePage 平台 Footer（REQ-035）**：靜態文字「由 OnePage 提供技術支援」+ 「聯絡我們」連結（新分頁）；與購物車內容有視覺區隔線

## 互動行為
- 即時更新總價
- 調整範圍：數量調整與刪除（不含修改包裝規格）
- 數量設為 0 等同刪除
- 純前端 Local Storage 實作
- 預購商品不可與一般商品混合
- 僅限同一網站商品
- 版面需對照 prototype/目錄下設計稿進行還原（REQ-031）
