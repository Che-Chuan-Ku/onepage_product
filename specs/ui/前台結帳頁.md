# 前台結帳頁

## 描述
顧客的下單表單頁面（/checkout）。匿名使用者不需登入。

## 佈局
- 訂單商品摘要
- 姓名欄位（必填）
- 地址欄位：縣市 + 區 + 詳細地址（純文字）
- 電話欄位（必填）
- Email 欄位（必填，格式驗證）
- 備註欄位（選填）
- 統一編號欄位（選填，8 碼數字驗證）
- 運送方式選擇：宅配 / 自取
- 運費顯示（即時計算）
- 送出訂單按鈕
- **OnePage 平台 Footer（REQ-035）**：靜態文字「由 OnePage 提供技術支援」+ 「聯絡我們」連結（新分頁）

## 互動行為
- 地址輸入後呼叫 API 即時計算運費
- 自取時運費 NT$0，免地址驗證
- 外島地址顯示「不支援外島運送」並禁用送出按鈕
- 滿額自動免運
- 表單前端即時驗證
