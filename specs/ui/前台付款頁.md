# 前台付款頁

## 描述
顧客選擇付款方式的單頁式付款頁。

## 佈局
- 訂單摘要（金額、商品）
- 付款方式選擇：LINE Pay / 轉帳 / 信用卡
- 發票資訊區：
  - 發票類型：二聯式（個人）/ 三聯式（公司）
  - 載具類型：手機條碼 / 自然人憑證
  - 載具號碼輸入
- 確認付款按鈕
- **OnePage 平台 Footer（REQ-035）**：靜態文字「由 OnePage 提供技術支援」+ 「聯絡我們」連結（新分頁）

## 互動行為
- 選擇付款方式後跳轉綠界
- 載具格式即時驗證
  - 手機條碼：/+7碼英數
  - 自然人憑證：2碼大寫英文+14碼數字
