# 後台商品管理頁

## 描述
管理員上架、編輯、下架商品的管理頁面。支援新增商品與編輯已存在商品資訊。

## 佈局
- 商品列表（含篩選）
- 新增商品表單：名稱、描述、價格（單位選擇：公斤/台斤）、圖片上傳（最多5張）、分類選擇、庫存、包裝規格（純文字）
- 編輯商品表單：與新增表單共用欄位（名稱、描述、價格、圖片、分類、包裝規格），但不含庫存欄位（庫存透過庫存管理頁編輯）
- 預購商品設定區：預購期間、專屬折扣百分比
- 組合包設定區：勾選包含商品 + 百分比折扣
- 商品狀態管理（上架/下架）

## 互動行為
- 填寫表單後提交上架
- 點擊商品列表項目進入編輯模式，修改後儲存
- 編輯商品名稱時 slug 重新產生
- 已上架與已下架商品皆可編輯
- 圖片拖拉上傳，支援 JPG/PNG/WebP，單張 2MB 上限
- 價格必須 > 0 即時驗證
- 分類為二級：主分類（水果類/南北乾貨）→ 子分類（進口/台灣產）
- 下架操作需確認
